Rules

General Rules

This is a formula Fantasy game where players predict the race results by answering the questions

User can submit filled qiuz for the next race from monday after last race till the saturday 23:59 before the next race. On the race day is the form locked.

It is possible to update the form by filling it again. As many times as you want.

Answers are evalated after the race at 23:00

Pointing system

For each question there is a set of possible answers. There are three groups of questions

A - This group of questions is required, player gets +2 point right answer and -1 point wrong answer

B - This group of questions is optional, player gets +2 point right answer and -1 point wrong answer

C - This group of questions is required player gets +2 point right answer and -1 point wrong answer

Points are awarded for each correct answer. The user with the most points wins.
